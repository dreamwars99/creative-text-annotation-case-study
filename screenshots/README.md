# Screenshots

Only sanitized portfolio images should be added here.

Do not add:

- full product screenshots,
- private UI flows,
- unreleased roadmap screens,
- user content,
- internal prompts,
- provider settings,
- or any image that reveals the private product architecture.

Recommended safe images:

- a cropped candidate review card,
- a simplified annotation flow diagram,
- a redacted schema preview,
- or a hand-made mock screenshot created only for this public case study.
